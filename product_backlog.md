# Product Backlog - Habit Tracker App

This backlog outlines the prioritized user stories and features required for the UVW Code Labs Habit Tracker project.

---

## High Priority (MVP - Minimum Viable Product)

### 1. Account Registration & Login
- **Description:** As a user, I want to create an account and log in so that I can access the habit tracker features.
- **Labels:** `enhancement`, `priority: high`

### 2. Habit Tracking Homepage
- **Description:** As a user, I want a dashboard to view my daily progress so that I can see my streaks at a glance.
- **Labels:** `enhancement`, `priority: high`

### 3. Application Navigation Menu
- **Description:** As a user, I want a hamburger menu so that I can navigate between Profile, Habits, and Reports.
- **Labels:** `enhancement`, `priority: high`

### 4. Account Sign Out
- **Description:** As a user, I want to sign out so that my session is cleared and my data remains secure.
- **Labels:** `enhancement`, `priority: high`

---

## Medium Priority (Core Functionality)

### 5. Add and Personalize Habits
- **Description:** As a user, I want to add new habits and assign them colors to customize my experience.
- **Labels:** `enhancement`, `priority: medium`

### 6. Edit Personal Profile
- **Description:** As a user, I want to edit my profile info so that my details stay up to date.
- **Labels:** `enhancement`, `priority: medium`

### 7. Habit Progress Reports
- **Description:** As a user, I want to see a weekly progress report to track my consistency over time.
- **Labels:** `enhancement`, `priority: medium`

---

## Low Priority (Future Enhancements)

### 8. Task Reminders
- **Description:** As a user, I want to set reminders so that I don't forget to complete my daily habits.
- **Labels:** `enhancement`, `priority: low`
